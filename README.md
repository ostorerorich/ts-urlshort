# URL Shortener
This is a URL shortener application made with TypeScript, TypeORM, and PostgreSQL. It takes a long URL and shortens it to make it easier to share.

## Installation
1. Clone the repository:
   ```
    git clone https://github.com/ostorerorich/ts-urlshort.git
   ```

2. Install Dependencies:
  ```
  cd ts-urlshort
  npm install

  ```

3. Set up your database connection:

    * Create a new PostgreSQL database for the application.
    * Copy the .env.example file to .env and update the values with your database information.
  
4. Transpile and run:
   ```
   npm run build && npm run start
   ```
